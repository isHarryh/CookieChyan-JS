# CookieChyan-JS

A Convenience Tool to Retrieve Cookies in Any Webpage  
一个方便的网页 Cookie 获取工具

## Introduction

This Tampermonkey user script is designed to help users easily retrieve cookies from any webpage.

With a keyboard shortcut <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>C</kbd>, users can quickly copy the cookies of the current page to their clipboard for further use.

## Development

### Debug with hot-reload

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
